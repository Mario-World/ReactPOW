## Getting Started

Each project is self-contained in its own folder.  
To run a project:

1. Navigate to the project folder:
   ```sh
   cd <project-folder>
   ```
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```
